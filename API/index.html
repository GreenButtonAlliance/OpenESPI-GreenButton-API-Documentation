Green Button API back to Services

 

The Green Button API provides access to RESTful services provided by Data Custodian and Third Party Applications. These APIs allow application developers full access, in Green Button Format, to Energy Usage Information for the purpose of developing new Green Button Applications.

Access to Green Button APIs normally require an OAuth2 Access Token. This API sandbox currently does not require a token. In the near future, the API entry's "Try it out!" button will require a valid access token for successful operation. When that occurs, directions will be provided and the token will need to be entered in the input field labelled "Access Token" below.
